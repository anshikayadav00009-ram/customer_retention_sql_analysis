# 🛒 E-Commerce Sales Data Analysis (SQL Project)

## 📌 Project Overview
This project focuses on analyzing e-commerce sales data using SQL to extract meaningful business insights. 
The goal is to understand sales trends, customer behavior, and product performance.

## 🎯 Problem Statement
E-commerce businesses generate large amounts of sales data. 
However, without proper analysis, it becomes difficult to understand:
- Which products are performing best?
- Who are the most valuable customers?
- What are the monthly revenue trends?
- Which categories generate maximum profit?

This project aims to solve these business questions using SQL queries.

## 🎯 Objectives
- Design a relational database for an e-commerce system
- Perform data analysis using SQL queries
- Use aggregations, joins, and grouping techniques
- Generate business insights from raw data
